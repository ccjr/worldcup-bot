worldcup-bot
============

An API for a hungrybot script

## Endpoints:

###### Today's Matches:

```
/matches
```

###### Tomorrow's Matches:

```
/matches/:country_short_name
```

**Example:**

```
/matches/USA
```

###### Today's Scores:

```
/scores
```

###### Team's Past/Current Scores:

```
/scores/:country_short_name
```

**Example:**

```
/matches/BRA
```
